Cow Cost App
========================
Application written in Symfony3 to calculate the cow cost integrated with API REST service.


Installation
--------------
Run in root folder

```
composer install 
```

Run in web folder
```
npm install 
```

Running
--------------
Run in root folder
```
php bin/console server:run
```

Dependencies
--------------
* composer
* node
* npm
* materialize-css
